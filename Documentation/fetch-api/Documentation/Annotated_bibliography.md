# Annotated bibliography



1. MDN Web Docs
    * URL: [MDN Web Docs: Fetch](https://developer.mozilla.org/en-US/docs/Web/API/Fetch_API/Using_Fetch)
    * This resource provide detailed information on the Fetch API and also has a comprehensive guide on using the Fetch API. The guide includes making GET & POST requests and handling responses and errors.

2. W3Schools
    * URL: [W3Schools: Javascript Fetch API Demo](https://www.w3schools.com/js/js_api_fetch.asp)
    * W3Schools offers a tutorial and a demo on the Fetch API. It includes a demo on make GET & POST requests, example code snippets as well as step-by-step instructions.

3. GeeksforGeeks
    * URL: [GeeksforGeeks: Article on Fetch API](https://www.geeksforgeeks.org/fetch-api/)
    * The url is a link to an article on using the fetch API to perform GET and POST requests, with examples and also discusses types of responses and some errors you can run into.

    * URL: [PHP and JSON](https://www.w3schools.com/php/php_json.asp)
    * This resource provides a demo on how to use `json_endcode` and `json_decode` which are built-in functions for handling JSON with PHP.

4. PHP Manual
    * URL: [PHP Manual](https://www.php.net/manual/en/langref.php)
    * This is the best resource to learn PHP as it is the official docummentation for PHP. The manual contains detailed information on everythin PHP, and since we are interested in how to handle GET and POST requests in PHP functions like `json_endcode`, `json_decode` and `file_get_contents` and predefined variables like `$_SERVER` come in really handy.

5. Tutorialspoint
    * URL: [How to receive JSON POST with PHP](https://www.tutorialspoint.com/how-to-receive-json-post-with-php#:~:text=Send%20the%20JSON%20data%20in,PHP%20associative%20array%20or%20object.)
    * This is a detailed article that covers receiving POST requests with PHP. It provides code snippets and comprehensive explanation.

6. YouTube
    * URL: [Web Dev Simplified](https://www.youtube.com/watch?v=cuEtnrL9-H0)
    * YouTube is an amazing resource for learning and  this creator has a very comprehensive video explaining how to implement and test the fetch API in javascript.
    
    * URL : [Digital Fox](https://www.youtube.com/watch?v=Ctz1Fsgt9OE)
    * This resources shows an example on how to handle request in PHP from Javascript fetch API.
